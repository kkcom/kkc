The KKD files ?

What is a KKD file.

KKD Files are a list of files and subdirectories. These files are a copy of the tree of a disk (removable for a best use) and they could be used for quick search of information without using the original disk.

How could I do a KKD file ?

Go to the root of the disk that you would create a KKD file. Press SHIFT-F12. Enter the name of the KKD file to be created. Wait... wait... wait. ¿ƒ It's finished ! In the other window, the KKD file exists...

How could I use a KKD file ?

When you are on a KKD file, press ENTER on it. You are now into the KKD file. The structure of the files is the same that on the original disk. You could access to a file in a KKD as any file on any disk. A menu ask you to enter the drive where is the orignal disk.

Questions and answers

The startup of KKC is too slow on my computer.

  • :( Here is a few tips to speed up KKC:
    • Select the Quick Palette Option. (On my P100, I win 5 clocks while the startup when this option is on. It's weird...this option only change the palette)
    • Unselect the 'Verify History Directory' in the Configuration.
    • Don't load 'emm386' in your config.sys (a ems driver slow dowm all the application what run in protected mode).
    • Load a Disk Cache Utility.
    • If you have a Pointing Device Port (PS/2...), unloading your mouse driver will speed up a lot the startup speed (The initialisation of the mouse with this port take a lot of time.)

I couldn't copy or view file in archive !

  • You must have the archiver on your disk ;).
  • KKC couldn't be in a too big directories.

I don't like the font.

  • Erase the file font8x16.cfg (in mode 25/30 lines) or font8x8.cfg (in mode 50 lines).
  • You could too press CTRL-F (to switch to normal fonts)
  • You could create and import new font with my program 'kkfont' kkfont is not included in the current version of KKC (it is not finished) but if you want its presence in a next version, you could ask to me !

ALT-F4 don't work.

  • Verify that the size of the user directory is greater than the the size of file_id.diz.
  • Verify that your archiver works.

When I run a player, there is an error with this message:

≥Stub exec failed:
≥dos4gw.exe
  • You must copy the 'DOS4GW.EXE' file in a directory accessible by the path.

Configuration

[X] Convert History

When this option is on, each command that you enter in command line is translated in a shorter form. So, when you call the history window, the command line is automatically translated in a new form depending on the files in the two panels.

[X] Confirmation before exiting

Ask if you really want to quit when press F10.

[ ] Point SubDir

Display the '.' directory (reload) in the file panel.

[ ] LogFile

Create a logfile of the command in the user directory.

[ ] ESC to close windows

You could press ESC to open/close the two file panel.

[ ] Display Hidden File

Display also the hidden file in the file panel.

[X] Auto Reload Directory

When modifying file, it reload the file panel to corresponding to the true contents of the directory. Set this option to on unless KKC run slowly on your system.

[ ] Verify History Directory

When starting KKC, it verify the directory list in the history.

[X] Quick Palette

When off, the palette is loaded during the starting of KKC. When on, the palette is loaded after all checking.

[X] Highlight groups

Highlight groups of files according to their extension. You could unset a group by modifying directly the list in 'Ext. Setup'.

[X] Insert move down

When you press insert, cursor go down.

[X] Select directories

When you select multiple files, the directories are also selectionned.

[X] Estimated copying time

When copying file, you have a estimation of the remaining time.

[X] Display lift

Display a lift at the right of each pannel.

[X] Save position in viewer

Save the viewing position when you quit the viewer.

[X] Load startup directory

Go in the current directory when you start KKC.

Size Trash

Maximum size of the trash before erasing temp files

[ ] Key After Shell

Wait for a keypress after a program's execution.

[ ] Add Selection Function

Add a selection function at the end of each pannel. Very useful for the notebook where there aren't numeric keypad.

Execute command

Execute command

Enter the command you will execute. All the result were redirected to an temporary file and are displayed with the internal viewer

You could recall the last command with CTRL-E.
Unless you have unset the 'Convert History' option, KKC translate the command line in a shorter form.

User Directory

When you run KKC for the first time, it creates a directory where it will copy all your personnal files (config, menu, temp...)
The default user directory is the 'trash' directory in your KKC directory.

You could modify the user directory with the environment variable KKUSER.
EG. KKC is in the 'c:\util\kkc' directory.
  • If you don't use the KKUSER variable, your u.d. is 'c:\util\kkc\trash'
  • If you use 'set KKUSER=redbug', your u.d. is 'c:\util\kkc\redbug'
  • If you use 'set KKUSER=c:\redbug', your u.d. is 'c:\redbug'
If you want put KKC on a CD, think to this variable :-)
Rem. Be careful the variable 'KKUSER' (with 'set KKUSER=redbug') is not the same that the variable 'KKUSER ' (with 'set KKUSER = redbug') !

The Safety Resolution

Automaticaly switches to a safety resolution:
  • text mode to 80 columns and 25 lines,
  • dos font,
  • dos palette,
  • NC Style's color,
  • Normal Output (nor ansi, nor doorway).
You could recall this function to restore the previous configuration.

Mouse support

If there is a mouse connected to your PC and a valid mouse driver is installed, KKC automatically uses the mouse.
The mouse support permit you to do everything you do with the keyboard (I hope...)

In a pannel

Left click on a file to go on it.
Left click on the current file to execute it.
Shift + left click to selection the file. (You could extend your selection if you don't release the left button)
Drag & Drop: you select a pannel and whithout releasing the button, you go to the other pannel. ( such with windows or OS/2 )

In the other place.

Normaly you use buttons and switches by pressing the left mouse button and the right button to cancel.

The optimized command line

TAB to get the next matching filename.
UP to search the previous matching command line in the history.
DOWN to search the next matching command line in the history.
